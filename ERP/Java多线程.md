# Java多线程

## Thread类

### 各种方法

- yield方法

	- 放弃当前CPU资源，让其他任务去占用CPU执行时间

### 优先级

- 概念

	- CPU让高优先级线程获取更多时间片

- 分级1-10

	- 默认是5，线程优先级和代码编写顺序无关，并不完全按优先级执行

### 守护线程

- 概念

	- 作用是为用户线程提供便利服务，如垃圾回收线程，
